Récupération des informations relatives à la présence d'enfant ou non¶

Les personnes s'enregistrant sur le site peuvent renseigner un champs "avec enfant(s)", on considrera que les personnes n'ayant pas renseign ce champs n'ont pas d'enfants. Cette aproche peut tre biais car certaines personnes ayant des enfants peuvent ne pas avoir renseign le champs.

In [6]:
def parse_enfant_from(user_info):
    if len(user_info) == 2:
        return 1
    else:
        return 0

df["enfant"] = df["user_info"].apply(parse_enfant_from)

def parse_date(date):
    import locale
    locale.setlocale(locale.LC_ALL, 'fr_FR')
    date_splited = date.split(' ')
    day = date_splited[0]
    year = date_splited[-1]
    month = date_splited[-2]
    if len(day) == 1:
        day = f'0{day}'
    return  f"{day} {month} {year}"

Conclusion¶

L'objectif de cette analyse tait d'explorer les commentaires d'un produit rcuprs via le web scrapping afin de dterminer quel type de personnes commentent (mme si l'chantillon n'est pas reprsentatif de l'ensemble du site). L'autre objectif tait d'ventuellement identifi des amliorations possibles en explorant les commentaires ngatifs. En ce qui concerne le type de personnes qui commentent, sur ce jeu de donnes il s'agit de femmes avec enfants ayant en moyenne 43 ans vivant dans les hauts de seine. En revanche, l'analyse des commentaires plutt ngatif n'a pas permis de tirer des amliorations.
